## How to Run Locally


Clone the repository:

```
git clone https://github.com/yourusername/roshanjha.com.git
```

Navigate into the directory:

```
cd roshanjha.com
```

Install nodejs dependencies:

```
npm install
```


Install Composer dependencies:

```
composer install
```

Start the development server:

```
php artisan serve
```
